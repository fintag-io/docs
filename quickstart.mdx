## Get started in three steps


### Step 1: Install the SDK

<CodeGroup>
  ```bash JavaScript
  npm install @fintag/js
  ```
  ```bash Python
  pip install fintag-py
  ```
  ```bash Rust
  cargo add fintag-rust
  ```
  ``` bash Go
  go get github.com/fintag-io/fintag-go
  ```
  ```bash Swift
  swift package add @fintag-io/fintag-swift
  ```

</CodeGroup>

### Step 2: Initialize the SDK

<CodeGroup dropdown>
  ```javascript JavaScript
  import { FintagClient } from '@fintag/js';

  const fintag = new FintagClient(
    apiUrl='YOUR_API_URL',
    apiKey='YOUR_API_KEY'
  );
  ```
  ```python Python
  from fintag import FintagClient

  fintag = FintagClient(
      api_url="YOUR_API_URL",
      api_key='YOUR_API_KEY'
  )
  ```
  ```rust Rust
  use fintag::FinTag;

  let fintag = FinTag::new("YOUR_API_KEY");
  ```
  ```go Go
  import (
      "github.com/fintag-io/fintag-go"
  )

  fintag := fintag.NewFintagClient("YOUR_API_KEY")
  ```
  ```swift Swift
  import FinTag;

  let fintag = FinTagClient(
      baseURL: "YOUR_API_URL",
      apiKey: "YOUR_API_KEY"
  )
  ```
</CodeGroup>

### Step 3: Make your first API call

<CodeGroup dropdown>
  ```javascript JavaScript
  const response = await fintag.verify("your-parameter");
  console.log(response);
  ```
  ```python Python
  response = fintag.verify("your-parameter")
  print(response)
  ```
  ```rust Rust
  let response = fintag.verify("your-parameter").await?;
  println!("{:?}", response);
  ```
  ```go Go
  response, err := fintag.Verify("your-parameter")
  if err != nil {
      log.Fatal(err)
  }
  fmt.Println(response)
  ```
  ```swift Swift
  do {
      let response = try await fintag.verify("your-parameter")
      print(response)
  } catch {
      print("Error: \(error)")
  }
  ```
</CodeGroup>


You're now ready to start building with FinTag!
For installation details and contribution setup, head over to the Usage section.

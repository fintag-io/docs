---
title: "Installation"
description: "Instructions for installing FinTag SDK"
---


FinTag provides official SDKs for various programming languages. Choose the one that best fits your stack.

<Tabs>
  
  <Tab title="JavaScript/TypeScript">
  <Info>
    **Prerequisites**:
    - Node.js version 19 or higher
  </Info>

  ### Install the SDK
  To install the FinTag SDK, use one of the following package managers:

  <CodeGroup>
  ```bash npm
  npm install @fintag/js
  ```
  ```bash yarn
  yarn add @fintag/js
  ```

  ```bash pnpm
  pnpm add @fintag/js
  ```

</CodeGroup>

  </Tab>

  <Tab title="Python">
  <Info>
    **Prerequisites**:
    - Python version 3.8 or higher
  </Info>

  <CodeGroup>
  ```bash pip
  pip install fintag-py
  ```

  ``` bash uv
  uv pip install fintag-py
  ```

  </CodeGroup>

  </Tab>

  <Tab title="Rust">
  <Info>
    **Prerequisites**:
    - Rust version 1.60 or higher
  </Info>

```bash
cargo add fintag-rust
```

  </Tab>

  <Tab title="Go">
  <Info>
    **Prerequisites**:
    - Go version 1.18 or higher
  </Info>

```bash
go get github.com/fintag-io/fintag-go
```

  </Tab>

  <Tab title="Swift">
  <Info>
    **Prerequisites**:
    - Swift version 5.5 or higher
  </Info>

```bash
swift package add @fintag-io/fintag-swift
```

  </Tab>

</Tabs>

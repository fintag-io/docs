## FinTag SDK Roadmap
Our roadmap outlines the planned features and improvements for the FinTag SDK. We are committed to continuously enhancing the SDK to meet the needs of our users and stay ahead in the financial technology space.

### Upcoming Features
(Q3 2025)
- Add support for additional programming languages (e.g., Java, C#)
- Enhanced documentation with more examples and use cases

(Q4 2025)
- Reverse Lookup from wallet address to fintag
- Enhanced security features and compliance updates
- Improved performance and scalability for high-volume applications

(2026 and Beyond)
- Cross-chain support for multiple blockchain networks asides Solana
- Sub-Tags (e.g., #developer.fintag) for organizations
- Analytics and reporting tools for fintag usage
- Verification badges (e.g., #verified.fintag) for users and organizations as an extra layer of trust
- “Login with FinTag” OAuth Flow

### Planned Improvements
- Regular updates to dependencies for security and performance
- Ongoing documentation enhancements based on user feedback
- Community-driven feature requests and contributions

### How to Stay Updated
Stay tuned to our [GitHub repository](https://github.com/fintag-io) for the latest updates and announcements.

We welcome contributions from the community. If you have ideas or would like to contribute, please refer to our [Contributing Guide](./contributing) for more information.

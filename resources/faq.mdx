### What is FinTag SDK?
FinTag SDK is a collection of tools and libraries that help developers integrate financial tagging and categorization features into their applications. It supports multiple programming languages including JavaScript, Python, Rust, Go, and Swift.

### How do I install FinTag SDK?
Please refer to the [Installation Guide](../usage/installation) for detailed instructions on how to install FinTag SDK for your preferred programming language.

### Where can I find the documentation for FinTag SDK?
You can find the complete documentation on our [official website](https://docs.fintag.io). The documentation includes guides, API references, and examples to help you get started.

### How do I get support for FinTag SDK?
If you encounter any issues or have questions, you can reach out to our support team via the [Support Page](../resources/support) or join our community forums for assistance from other developers.

### Can I contribute to FinTag SDK?
Yes! We welcome contributions from the community. Please check out our [Contributing Guide](../resources/contributing) for more information on how to get involved.

### What is the license for FinTag SDK?
FinTag SDK is licensed under the MIT License. For more details, please refer to the [License Page](../resources/license).


### What are the future plans for FinTag SDK?
We are continuously working on enhancing FinTag SDK with new features and improvements. You can check out our [Roadmap](../the-future/roadmap) for upcoming features and planned releases.

### How do I report a bug or request a feature?
You can report bugs or request new features by opening an issue on our [GitHub repository](https://github.com/fintag-sdk/issues).
